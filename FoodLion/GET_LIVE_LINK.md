## 🆘 IF SOMETHING GOES WRONG

### "502 Bad Gateway" Error
- Check environment variables are set correctly
- Verify SECRET_KEY and JWT_SECRET are in the form
- Check build logs in Railway dashboard

### "Deploy Failed" Error
- Click "View Logs" to see what went wrong
- Common: Missing environment variable
- Solution: Add all 5 environment variables

### "ModuleNotFoundError" Error
- Means a Python package is missing
- Solution: Check requirements.txt is in backend folder

### App Not Loading
- Wait 5 minutes - deployment might still in progress
- Refresh the page (Ctrl+F5)
- Check Railway dashboard for errors

**For detailed troubleshooting, see DEPLOYMENT_GUIDE.md**
