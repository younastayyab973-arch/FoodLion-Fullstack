Railway will show a form. Add these 5 variables:

```

ALLOWED_HOSTS=*.railway.app
CORS_ALLOWED_ORIGINS=https://yourdomain.railway.app
```

(Note: Railway auto-generates DATABASE_URL, so you don't need to add it)
