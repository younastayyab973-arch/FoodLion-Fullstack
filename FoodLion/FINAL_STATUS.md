## 🚀 For Production

When ready to deploy:

1. **Collect static files:**
   ```bash
   python manage.py collectstatic
   ```

2. **Use gunicorn:**
   ```bash
   gunicorn foodlion.wsgi
   ```

3. **Deploy to Railway.app:**
   - Push to GitHub
   - Railway auto-deploys
   - Visit your live link
